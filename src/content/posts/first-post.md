# 안녕하세요! 블로그에 오신 것을 환영합니다.

이 블로그는 **OSINT** 테마로 제작되었습니다.

## 주요 기능
- 마크다운 기반 글 작성
- 날짜순 정렬
- 태그 및 카테고리 지원

### 시스템 로그
```bash
$ npm install gray-matter react-markdown
$ mkdir content/posts
```

앞으로 유용한 정보를 이곳에 공유하겠습니다.
